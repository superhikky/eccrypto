class  ecc::PseudoRandomBitGenerator
 擬似ランダムビット生成器を表す。 [詳解]
 
class  ecc::RandomNumberGenerator< V, RBG >
 乱数生成器を表す。 [詳解]
 

詳解

このモジュールには乱数を生成するためのクラスが含まれる。
ecc::RandomNumberGeneratorは乱数を生成するためにランダムビット生成器を用いる。
既定のランダムビット生成器はecc::PseudoRandomBitGeneratorである。
ecc::PseudoRandomBitGeneratorはメルセンヌ・ツイスタ19937による擬似ランダムビットを生成する。
ecc::RandomNumberGeneratorの2番目のテンプレート引数にランダムビット生成器の型を渡すことで生成方法を選ぶことができる。
