詳解

template<typename V>
class ecc::rational_point< V >

楕円曲線上の有理点を表す。

無限遠点フラグ、座標、生成源であるecc::EllipticCurveへのバックポインタを保持する。
座標はヤコビアン座標系(x, y, z)である。
アフィン座標系(X, Y)とは次のような関係になる。

\( (x, y, z) = ( \frac{x}{z^2}, \frac{y}{z^3}, 1) = (X, Y, 1) \)

テンプレート引数
V値の型(整数型)。
