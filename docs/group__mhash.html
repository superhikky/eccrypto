詳解

このモジュールには平文のハッシュ値を計算するための関数が含まれる。
これらの関数はecc::InputOutputProcessに渡すための処理関数を作る。

関数詳解

◆ sha1()

number_to_number< uint32_t, uint32_t > ecc::sha1 ( )

SHA-1でハッシュ値を計算する処理関数を作る。

メッセージはブロック単位で入力する。
ブロックの長さは512ビット(4バイト×16)である。
出力するハッシュ値は160ビット(4バイト×5)である。

戻り値
作った処理関数。
警告
メッセージの末尾にはメッセージ全体の長さをビット長で付加しなければならない。
