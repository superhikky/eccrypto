template<typename V, class RBG = PseudoRandomBitGenerator>
class ecc::RandomNumberGenerator< V, RBG >

乱数生成器を表す。

任意の整数型の乱数を生成する。

テンプレート引数
V値の型(整数型)。
RBGランダムビット生成器の型。既定値はecc::PseudoRandomBitGenerator。
警告
RBGはデフォルトコンストラクタ、getBitメンバ関数を実装していなければならない。
